## 📋 Sobre o Projeto

Site institucional desenvolvido para o escritório **Vasconcellos & Amadeo Advocacia**, localizado em Vitória – ES, com atendimento em todo o Brasil.

O projeto foi construído com foco em **conversão**, **velocidade** e **experiência do usuário**, transmitindo confiança e profissionalismo desde o primeiro acesso.

---

## ✨ Funcionalidades

- 🎯 **Hero section** com CTA direto para WhatsApp
- ⚖️ **Áreas de atuação** — Direito Previdenciário e Direito do Trabalho
- 👨‍⚖️ **Perfil da equipe** — Dra. Genaina Vasconcellos e Dr. Rodolfo Amadeo
- ⭐ **Depoimentos reais** de clientes satisfeitos
- 📍 **Localização** com mapa integrado do Google Maps
- 📱 **100% responsivo** — menu hambúrguer no mobile
- 💬 **Botão flutuante do WhatsApp** com animação pulse
- 🎨 **Animações de scroll** suaves com Intersection Observer

## 🚀 Como usar

Basta abrir o arquivo `index.html` no navegador — sem dependências, sem build, sem instalação.

```bash
# Clone o repositório
git clone https://github.com/frandony/site-advocacia.git

# Abra no navegador
start index.html
```
